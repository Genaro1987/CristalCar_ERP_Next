# Convenção de códigos de tela (RH e demais módulos)

Para manter consistência e facilitar a descoberta das telas:

- **CAD**: prefixo para telas de cadastro de dados acessórios (ex.: empresa, departamento, jornada, funcionário, perfis de acesso).
- **LAN**: prefixo para telas de lançamento operacional (ex.: LAN001_RH_PONTO e futuros lançamentos financeiros ou operacionais).

Sempre reutilize esses prefixos ao criar novas telas, evitando códigos divergentes como o caso histórico de `CAD007_RH_PONTO` versus `LAN001_RH_PONTO`.
